What we do:

We provide a high-pressure environment for C programmers to sharpen their skills. Every 5 minutes, the world gets a new problem. You have 3 minutes to solve it.

Rules & Regulations:

  • The 3-Minute Rule: You must submit your code within 3 minutes of the question appearing. After that, the system locks.
  • The Correctness Check: Points are only awarded if your code contains the required logic/keywords for the task.
  • Anti-Cheat: If you switch browser tabs or minimize the window, your current code buffer will be wiped.
  • Integrity: Write your own code. No AI-pasting.
